# cabs
Analyze taxi trajectories.

# data
Cab mobility traces are provided by the Exploratorium - the museum of science, art and human perception through the cabspotting project: http://cabspotting.org.
You can also use this data set of cab mobility traces that were collected in May 2008.
There are total 536 trajectory data files.
